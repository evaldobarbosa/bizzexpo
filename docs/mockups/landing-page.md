## 5. Considerações de Design

### Cores
- Definir paleta de cores do BizzExpo
- Landing do evento: usar cores do evento (definidas pelo organizador)

### Tipografia
- Font principal: Inter ou similar (moderna, legível)
- Hierarquia clara: H1 > H2 > H3 > body

### Responsividade
- Mobile-first
- Breakpoints: 640px, 768px, 1024px, 1280px

### Performance
- Lazy loading de imagens
- Otimização de fontes
- Core Web Vitals

### Acessibilidade
- Contraste adequado
- Alt text em imagens
- Navegação por teclado
- ARIA labels
